함수 Methods

Name Arguments Type Return Description
data N/A N/A array[json object]|jquery object[array[json object]]

컴포넌트에 바인드 되어 있는 가장 최근 상태의 데이터를 반환 합니다.

Returns the data of the most recent state bound to the component.
selFlag boolean|string N/A

반환되는 데이터의 타입을 결정하거나 데이터를 필터링 하기 위한 구분값 입니다.

It is a division value to determine the type of data returned or to filter the data.

selFlag 인자 값에 따라 다음과 같은 데이터를 반환 합니다.

The following data is returned according to the value of the selFlag argument.

  • undefined(selFlag 옵션이 설정되지 않은 경우) : array[json object] 타입의 데이터를 반환 합니다.
  • undefined(When the selFlag option is not set) : Returns data of type array [json object].

  • false : 컴포넌트에 바인딩 되어 있는 jquery object[array[json object]] 타입의 원래 유형의 데이터를 반환 합니다.

    false : Returns the data of the original type of the jquery object[array[json object]] type bound to the component.

    data 메서드로 가져온 데이터를 다른 데이터 관련 컴포넌트에 바인딩 할 때는 반드시 "false" 로 설정하여 원래 유형의 데이터를 바인딩 해야 양방향 데이터 바인딩이 활성 화 됩니다.

    When binding data imported by the data method to other data-related components, you must set "false" to bind the original type of data to enable two way databinding.

  • "selected" : 선택 된 노드의 데이터를 반환 합니다(array[object] 타입).
  • "selected" : Returns the data of the selected node(array[object] type).
  • "checked" : 체크 된 노드의 데이터를 반환 합니다(array[object] 타입).
  • "checked" : Returns the data of the checked node(array[object] type).
  • "checkedInLastNode" : 체크된 모든 요소들의 마지막 노드의 데이터를 반환 합니다(array[object] 타입).
  • "checkedInLastNode" : Returns the data of the last node of all checked elements(array[object] type).

arguments[1],

arguments[2],

...arguments[n]

 string N/A

data 메서드의 두번째 인자부터 n 번째 인자까지 데이터의 프로퍼티명을 인자로 지정하면 지정한 프로퍼티 값만 추출한 객체를 반환 합니다.

If you specify the property name of the data from the second argument to the nth argument of the data method as an argument, the object that extracts only the specified property value is returned.

var listInst = N([]).tree(".context")
    .bind([{ col01 : "", col02 : "", col03 : "", col04 : "", col05 : "", col06 : "" }]);
treeInst.data("checked", "col01", "col02", "col03");
    // [{ col01 : "", col02 : "", col03 : "" }]

첫번째 인자를 "checked" 나 "checkedInLastNode" 로 설정 했을 때만 작동 됩니다.

It works only when the first argument is set to "checked" or "checkedInLastNode".
context N/A N/A jquery object

context 요소를 반환 합니다.

Returns the context element.
selector string N/A

jQuery selector 구문을 입력하면 context 요소에서 지정한 요소를 찾아서 반환 합니다.

If you input jQuery selector syntax, the element specified in the context element is found and returned.
bind N/A N/A N.tree

컴포넌트에 데이터를 바인딩 하여 context 옵션으로 지정한 요소 안에 Tree 를 생성 합니다.

Creates the Tree in the element specified by context option by binding data to component.
data string N/A

바인딩 할 데이터를 지정 합니다.

Specifies the data to be bound.
select N/A N/A N.tree

노드를 선택 하거나 선택 된 노드 값을 반환 합니다.

Selects the node or return the selected node value.

인자가 없으면 선택 된 노드 값을 가져오고 첫번째 인자를 입력하면 입력된 값으로 노드를 선택 합니다.

If there is no argument, selected node value is got, and if the first argument is entered, selects the node as the entered value.
val string N/A

선택 할 노드 값을 입력 합니다.

Enter the node value to select.
openAll N/A N/A N.tree

모든 트리 노드를 펼칩니다.

Expands all tree nodes.
closeAll N/A N/A N.tree

모든 트리 노드를 접습니다.

Collapse all tree nodes.
isFirstNodeOpen boolean N/A

true로 설정하면 첫 번째 노드를 제외한 모든 노드가 접힙니다.

If set to true, all nodes except the first node are collapsed.
