유틸리티 Utility

Name Arguments Type Return Description
N.doc.createLoadIndicator N/A N/A N.docs

원하는 시점에 로딩바와 화면클릭을 차단하는 요소를 생성 할 수 있습니다.

다음과 같이 호출하여 N.doc.createLoadIndicator 함수의 this 를 생성된 N.docs 인스턴스로 지정 합니다.

var docsInst = N("#docsContainer").docs();
...
N.docs.createLoadIndicator.call(docsInst);

N.doc.createLoadIndicator 와 N.doc.updateLoadIndicator, N.doc.removeLoadIndicator 함수를 override 하면 서버로 요청 시 자동으로 표시되는 로딩바와 화면클릭을 차단하는 요소를 다른 스타일로 표시 할 수 있습니다.
N.doc.updateLoadIndicator N/A N/A N.docs

원하는 시점에 로딩바의 진행률을 업데이트 할 수 있습니다.

다음과 같이 호출하여 N.doc.updateLoadIndicator 함수의 this 를 생성된 N.docs 인스턴스로 지정 합니다.

var docsInst = N("#docsContainer").docs();
...
N.docs.createLoadIndicator.call(docsInst);
...
...
N.docs.updateLoadIndicator.call(docsInst, 2, 2);
...
N.docs.updateLoadIndicator.call(docsInst, 1, 2);
...
N.docs.updateLoadIndicator.call(docsInst, 0, 2);
...

N.doc.createLoadIndicator 와 N.doc.updateLoadIndicator, N.doc.removeLoadIndicator 함수를 override 하면 서버로 요청 시 자동으로 표시되는 로딩바와 화면클릭을 차단하는 요소를 다른 스타일로 표시 할 수 있습니다.
entireLoadRequestCnt number N/A

남은 진행 프로세스 수

완료 될 때 까지 1씩 차감 합니다.
entireLoadRequestMaxCnt number N/A

전체 진행 프로세스 수
N.doc.removeLoadIndicator N/A N/A N.docs

>원하는 시점에 로딩바와 화면클릭을 차단하는 요소를 제거(Hide) 할 수 있습니다.

다음과 같이 호출하여 N.doc.removeLoadIndicator 함수의 this 를 생성된 N.docs 인스턴스로 지정 합니다.

var docsInst = N("#docsContainer").docs();
...
N.docs.createLoadIndicator.call(docsInst);
...
N.docs.updateLoadIndicator.call(docsInst, 2, 2);
...
N.docs.updateLoadIndicator.call(docsInst, 1, 2);
...
N.docs.updateLoadIndicator.call(docsInst, 0, 2);
...
...
N.docs.removeLoadIndicator.call(docsInst);

N.doc.createLoadIndicator 와 N.doc.updateLoadIndicator, N.doc.removeLoadIndicator 함수를 override 하면 서버로 요청 시 자동으로 표시되는 로딩바와 화면클릭을 차단하는 요소를 다른 스타일로 표시 할 수 있습니다.
