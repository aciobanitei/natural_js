N.validator N/A N/A N.validator N.validator 객체를 생성 합니다.
obj object N/A 검증할 데이터세트를 지정 합니다. 데이터는 array[json object] 타입으로 입력 해야 합니다.
rules object|jquery object(elements) N/A

룰은 다음과 같이 두가지 타입으로 지정 가능 합니다.

* 라이브러리 초기화 시 명령형 옵션으로 지정 : 

// { "컬럼명" : [[룰명, arg0, arg1...], [룰명, arg0, arg1...]] }
{
    "numeric" : [["required"], ["integer+commas"]],
    "generic" : [["required"], ["korean"]],
    "limit" : [["required"], ["alphabet"]]
}

* data-validate 속성에 선언형 옵션으로 지정 : 

<input id="numeric" type="text"
    data-validate='[["required"], ["integer+commas"]]'/>
<!-- [ ["룰명", "룰arguments[0]", "룰 arguments[1]"], ... ] -->

선언형으로 검증 룰 지정 시 동적으로 검증 룰을 변경 하려면 다음과 같이 해당 입력 요소에 "validate" 라는 속성에 담겨있는 data(rules)를 변경 하면 됩니다.

N("#numeric").data("validate", [["required"], ["integer"]])
N(obj).validator N/A N/A N.validator

N.validator의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

Create an object instance of N.validator with jQuery plugin method.

var validator = N(data).validator(rules);

객체 인스턴스를 생성하는 방식은 다르지만 new N.validator() 로 생성한 인스턴스와 N().validator 로 생성한 인스턴스는 동일 합니다. N() 함수의 첫번째 인자가 new N.validator 생성자의 첫 번째 인자로 설정 됩니다.

The method of creating object instances is different, but the instance created with newvalidator and the instance created with validator are the same. The first argument of the N () function is set as the first argument of the new N.validator constructor.
rules object|jquery object(elements) N/A

N.validator 함수의 두번째 인자(rules)와 같습니다.

Same as the second argument(rules) of the N.formatter function.
