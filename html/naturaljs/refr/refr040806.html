Name Arguments Type Return Description
data N/A N/A array[json object]|jquery object[array[json object]]

컴포넌트에 바인드 되어 있는 가장 최근 상태의 데이터를 반환 합니다.

Returns the data of the most recent state bound to the component.
selFlag boolean|string N/A

반환되는 데이터의 타입을 결정하거나 데이터를 필터링 하기 위한 구분값 입니다.

It is a division value to determine the type of data returned or to filter the data.

selFlag 인자 값에 따라 다음과 같은 데이터를 반환 합니다.

The following data is returned according to the value of the selFlag argument.

  • undefined(selFlag 옵션이 설정되지 않은 경우) : array[json object] 타입의 데이터를 반환 합니다.
  • undefined(When the selFlag option is not set) : Returns data of type array [json object].

  • false : 컴포넌트에 바인딩 되어 있는 jquery object[array[json object]] 타입의 원래 유형의 데이터를 반환 합니다.

    false : Returns the data of the original type of the jquery object[array[json object]] type bound to the component.

    data 메서드로 가져온 데이터를 다른 데이터 관련 컴포넌트에 바인딩 할 때는 반드시 "false" 로 설정하여 원래 유형의 데이터를 바인딩 해야 양방향 데이터 바인딩이 활성 화 됩니다.

    When binding data imported by the data method to other data-related components, you must set "false" to bind the original type of data to enable two way databinding.

  • "modified" : 추가, 수정, 삭제 된 데이터를 반환 합니다(array[json object] 타입).
  • "modified" : Returns data that has been inserted, updated, or deleted.(array[json object] type).
  • "selected" : select 옵션이나 multiselect 옵션을 설정 했을 때 선택된 데이터를 반환 합니다(array[json object] 타입).
  • "selected" : Returns data selected when the select or multiselect option is set(array[json object] type).
  • "checked" : checkAll 옵션과 checkAllTarget 옵션 이나 checkOnlyTarget 옵션을 설정 했을 때 체크된 데이터를 반환 합니다(array[json object] 타입).
  • "checked" : Checked data is returned when the checkAll option and checkAllTarget option or checkOnlyTarget option are set(array[json object] type).
  • "insert" : 추가된 데이터를 반환 합니다(array[json object] 타입).
  • "insert" : Returns the inserted data(array[json object] type).
  • "update" : 수정된 데이터를 반환 합니다(array[json object] 타입).
  • "update" : Returns the updated data(array[json object] type).
  • "delete" : 타입의 삭제된 데이터를 반환 합니다(array[json object] 타입).
  • "delete" : Returns the deleted data(array[json object] type).

 string N/A

data 메서드의 두번째 인자부터 n 번째 인자까지 데이터의 프로퍼티명을 인자로 지정하면 지정한 프로퍼티 값만 추출한 객체를 반환 합니다.

If you specify the property name of the data from the second argument to the nth argument of the data method as an argument, the object that extracts only the specified property value is returned.

var listInst = N([]).list(".context")
    .bind([{ col01 : "", col02 : "", col03 : "", col04 : "", col05 : "", col06 : "" }]);
listInst.data("modified", "col01", "col02", "col03");
    // [{ col01 : "", col02 : "", col03 : "" }]

첫번째 인자를 "modified", "selected", "checked", "insert", "update", "delete" 로 설정 했을 때만 작동 됩니다.

It works only when the first argument is set to "modified", "selected", "checked", "insert", "update" or "delete".
context N/A N/A jquery object

context 요소를 반환 합니다.

Returns the context element.
selector string N/A

jQuery selector 구문을 입력하면 context 요소에서 지정한 요소를 찾아서 반환 합니다.

If you enter jQuery selector syntax, the element specified in the context element is found and returned.
contextBodyTemplate N/A N/A jquery object

리스트 bind 전 초기화 시 지정된 리스트 바디 템플릿 요소를 반환 합니다.

N.list 는 이 요소를 복제하여 행들을 만듭니다.

contextBodyTemplate 메서드를 통해 반환 된 요소의 구성을 수정하고 다시 바인드 하면 수정된 모양의 리스트 바디가 만들어 집니다.

동적으로 리스트 바디의 모양을 바꿔야 된다면 contextBodyTemplate 메서드와 jQuery 를 활용하여 템플릿을 수정하기 바랍니다.
selector string N/A selector 문자열을 입력하면 리스트 바디 템플릿 요소에서 선택된 요소를 반환 합니다.
bind N/A N/A N.list

context 옵션으로 지정한 요소 안에서 id 속성값을 갖고 있는 요소들에 데이터를 바인딩 하고 data 의 길이만큼 행 요소를 생성 합니다.

Binds data to elements that have id attribute value in the element specified by context option and creates the row elements as long as the data length.
data array[json object] N/A

바인딩 할 데이터를 지정 합니다.

Specifies the data to be bound.
"append" string N/A

"append" 문자열을 인자로 설정 하여 바인딩 하면 이전에 바인드 된 데이터와 새로 바인드 한 데이터를 병합하고 이전에 생성된 행 요소에 새로운 행 요소를 추가 해 줍니다.

When binding by setting the "append" string as an argument, it merges the previously bound data with the newly bound data and appends the new row element to the previously created row element.

"append" 인자 값이 설정 되면 scrollPaging.size 옵션 값 자동으로 0 으로 지정 되어 스크롤 페이징 기능이 비활성화 됩니다.

When the "append" argument value is set, the scrollPaging.size option value is automatically set to 0 to disable the scroll paging feature.
add N/A N/A N.list

새로운 로우 데이터를 생성 합니다.

로우 데이터 생성 시 리스트요소안의 입력요소의 id 와 값들로 데이터를 생성합니다.
data object N/A 새로운 행을 생성할 때 컬럼값들의 기본값을 지정 할 수 있습니다.

인자로 지정된 데이터는 add 호출 시 생성된 기본 데이터를 확장 합니다.

data 인자의 type 이 number 타입이면 다음 옵션인 row 로 인식 됩니다.
row object N/A

새로운 데이터를 추가 할 행 인덱스를 지정 합니다. 지정한 행 인덱스 앞에 행 요소와 데이터가 추가 됩니다.
remove N/A N/A N.list 선택한 행 요소와 데이터를 삭제 합니다.
row number|array N/A 제거할 행 인덱스를 지정 합니다. index 들을 array에 담아 인자로 지정 하면 한번에 여러 행을 제거 합니다.
revert N/A N/A N.list bind 했을 때의 초기 데이터나 add 시 생성 된 초기 데이터 상태로 복원 합니다.
row number|array N/A

row index를 입력하면 해당 행 데이터만 복원 하고 입력하지 않으면 전체 행 데이터를 복원 합니다.

index 들을 array에 담아 인자로 지정 하면 한번에 여러 행을 복원 합니다.
validate N/A N/A N.list

추가 / 수정 된 전체 데이터에 대한 유효성 검증 결과를 반환 합니다. 유효성 검증 성공 시 true 를 반환 하고 실패 시 false 가 반환 되면서 해당 입력요소 옆에 툴팁으로 실패 메시지를 표시 합니다.

Returns the result of validation on all added / modified data. If validation succeeds, it returns true, and on failure, false is returned, and a tooltip is displayed next to the input element.
row number N/A row index를 입력하면 해당 행데이터만 검증하고 입력하지 않으면 전체 데이터를 검증합니다.
val N/A N/A value object|N.list

선택한 컬럼의 원래 값을 가져오거나 입력(수정) 합니다.

val 인자를 입력하지 않으면 선택한 값을 반환하고 입력하면 선택한 키 컬럼의 값을 입력한 값으로 바꿔 줍니다.

값 입력 시 입력한 키 에 해당하는 데이터 없으면 새로운 컬럼 데이터를 생성 합니다.
row number N/A 가져올 값의 row index
key string N/A 가져올 값의 key 컬럼명
val string | number | boolean | null N/A

저장할 값

key 인자 값과 같은 id 속성값을 가진 HTML 요소가 있다면 null 은 빈 문자열 숫자는 숫자 문자열, Boolean 은 "true" 나 "false" 문자열로 변환 되어 저장 되고 대상 HTML 요소가 없다면 원형 그대로 저장 됩니다.
select N/A N/A array

그리드 행을 명령으로 선택 할 수 있습니다.

첫번째 인자(row)를 입력하지 않으면 선택 된 행 들의 행 index 들을 반환 하고 입력하면 입력 된 인덱스에 해당하는 행을 선택 합니다.

select 메서드를 사용 하려면 select 나 multiselect 옵션이 true 로 설정 되어 있어야 합니다.
row array|number N/A

선택할 row index

  • 행 1개를 선택 하려면 인자를 number 타입 으로 지정해도 되고 array 타입 으로 지정해도 됩니다.
  • 행을 2개 이상 선택 하려면 인자를 array 타입 으로 지정해야 합니다.
isAppend boolean N/A

true 를 입력 하면 이전에 선택 된 행들을 선택 취소 하지 않고 추가로 선택 합니다. false 를 입력 하면 이전에 선택 된 행들을 모두 선택 취소 후 선택 합니다.
check N/A N/A string

checkAllTarget 과 checkSingleTarget 옵션으로 지정한 체크박스 들을 명령으로 선택 할 수 있습니다.

  • 첫번째 인자(row)를 입력하지 않으면 선택 된 체크박스 가 속해 있는 행 index 들을 반환 하고 입력하면 입력 된 인덱스에 속해 있는 체크박스들을 선택 합니다.
row array|number N/A

체크할 체크박스들의 row index

  • 체크박스 1개를 선택 하려면 인자를 number 타입 으로 지정해도 되고 array 타입 으로 지정해도 됩니다.
  • 체크박스를 2개 이상 선택 하려면 인자를 array 타입 으로 지정해야 합니다.
isAppend boolean N/A

true 를 입력 하면 이전에 선택 된 체크박스 들을 선택 취소 하지 않고 추가로 선택 합니다. false 를 입력 하면 이전에 선택 된 체크박스 들을 모두 선택 취소 후 선택 합니다.
move N/A N/A N.list

이동 할 위치의 행 인덱스를 입력 합니다.

입력 한 값이 바인드 되어 있는 데이터의 전체 행 수 보다 클 경우 항상 마지막 행 다음으로 이동 됩니다.
