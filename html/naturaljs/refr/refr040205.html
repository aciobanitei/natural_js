선언형옵션 Declarative Options

N.button의 선언형옵션은 요소의 data-opts 속성에 설정하는 옵션입니다.

The declarative option of N.button is set in the element's data-opts attribute.

<a data-opts='{ "color" : "white", "size" : "big" }'>a</a>
<input type="button" value="input button" data-opts='{ "color" : "gray" }'>
<button data-opts='{ "color" : "green", "size" : "large" }'>button</button>

선언형옵션은 JSON 표준 형식을 정확하게 준수해야 합니다(프로퍼티 명은 반드시 큰 따옴표로 감싸야합니다). JSON 표준 포멧을 준수하지 않으면 선언형 옵션이 인식되지 않거나 오류가 발생합니다.

Declarative options must conform exactly to the JSON standard format(Property names must be enclosed in double quotes). If you do not comply with the JSON standard format, the declarative options are not recognized or an error occurs.

선언형 옵션에 대한 설명은 API 문서 안내 메뉴의 선언형옵션 섹션을 참고하세요.

For a description of the declarative options, see the Declarative Options section of the API Documentation Guide menu.

Name Type Default Description
size string "medium"

버튼의 크기를 설정합니다.

Sets the size of the button.

"smaller", "small", "medium", "large", "big" 중 하나를 지정할 수 있습니다.

You can specify "smaller", "small", "medium", "large" or "big".
color string "white"

버튼의 색상을 설정합니다.

Sets the color of the button.

"white", "blue", "skyblue", "gray", "green", "yellowgreen" 중 하나를 지정할 수 있습니다.

You can specify "white", "blue", "skyblue", "gray", "green", or "yellowgreen".
disable boolean false

true로 설정하면 버튼이 비활성화된 상태로 생성됩니다.

If set to true, the button will be created with disabled.
