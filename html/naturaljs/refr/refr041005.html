함수 Methods

Name Arguments Type Return Description
data N/A N/A array[json object]|jquery object[array[json object]]

컴포넌트에 바인드 되어 있는 가장 최근 상태의 데이터를 반환 합니다.

Returns the data of the most recent state bound to the component.
selFlag boolean N/A

반환되는 데이터의 타입을 결정하는 구분값 입니다.

This is a division value that determines the type of data returned.

selFlag 인자 값에 따라 다음과 같은 데이터를 반환 합니다.

The following data is returned according to the value of the selFlag argument.

  • undefined(selFlag 옵션이 설정되지 않은 경우) : array[json object] 타입의 데이터를 반환 합니다.
  • undefined(When the selFlag option is not set) : Returns data of type array [json object].

  • false : 컴포넌트에 바인딩 되어 있는 jquery object[array[json object]] 타입의 원래 유형의 데이터를 반환 합니다.

    false : Returns the data of the original type of the jquery object[array[json object]] type bound to the component.

    data 메서드로 가져온 데이터를 다른 데이터 관련 컴포넌트에 바인딩 할 때는 반드시 "false" 로 설정하여 원래 유형의 데이터를 바인딩 해야 양방향 데이터 바인딩이 활성 화 됩니다.

    When binding data imported by the data method to other data-related components, you must set "false" to bind the original type of data to enable two way databinding.
context N/A N/A jquery object

context 요소를 반환 합니다.

Returns the context element.
selector string N/A

jQuery selector 구문을 입력하면 context 요소에서 지정한 요소를 찾아서 반환 합니다.

If you enter jQuery selector syntax, the element specified in the context element is found and returned.
bind N/A N/A N.pagination

context 옵션으로 지정한 요소에 데이터를 바인딩 하여 Pagination 을 생성 합니다.

Pagination is created by binding data to the element specified by the context option.
data|totalCount array[object]|number N/A

바인딩 할 data를 지정 하거나 totalCount 를 설정 합니다.

Specifies the data to be bound or set totalCount.

인자 타입이 number 이면 totalCount 로 설정 되고 array 타입이면 data 로 지정 됩니다.

If the argument type is number, it is set to totalCount, and if it is an array type, it is specified as data.
totalCount number N/A 페이징 될 전체 로우 카운트가 다시 지정 됩니다.
pageNo N/A N/A number|N.pagination

페이지번호 값을 가져오거나 지정 합니다.

인자가 없으면 pageNo 값을 가져오고 첫번째 인자를 입력하면 입력된 값으로 pageNo가 지정 됩니다.

인스턴스 내부의 값만 변경 되므로 이후 bind 메서드를 호출 해 줘야 바뀐 부분이 화면에 표시됩니다.

N.pagination 이 이미 인스턴스화 된 후 페이지를 변경할때 사용 됩니다.
pageNo number N/A 입력된 값으로 컴포넌트 옵션의 pageNo가 지정 됩니다.
totalCount N/A N/A number|N.pagination

전체 행 건수 값을 가져오거나 지정 합니다.

인자가 없으면 totalCount 값을 가져오고 첫번째 인자를 입력하면 입력된 값으로 totalCount가 지정 됩니다.

인스턴스 내부의 값만 변경 되므로 이후 bind 메서드를 호출 해 줘야 바뀐 부분이 화면에 표시됩니다.

N.pagination 이 이미 인스턴스화 된 후 전체 행 수를 변경할때 사용 됩니다.
totalCount number N/A 입력된 값으로 컴포넌트 옵션의 totalCount가 지정 됩니다.
countPerPage N/A N/A number|N.pagination

페이지 당 행 수 값을 가져오거나 지정 합니다.

인자가 없으면 countPerPage 값을 가져오고 첫번째 인자를 입력하면 입력된 값으로 countPerPage가 지정 됩니다.

인스턴스 내부의 값만 변경 되므로 이후 bind 메서드를 호출 해 줘야 바뀐 부분이 화면에 표시됩니다.

N.pagination 이 이미 인스턴스화 된 후 페이지당 행수를 변경할때 사용 됩니다.
countPerPage number N/A 입력된 값으로 컴포넌트 옵션의 countPerPage가 지정 됩니다.
countPerPageSet N/A N/A number|N.pagination

페이지 세트 당 페이지 수 값을 가져오거나 지정 합니다.

인자가 없으면 countPerPageSet 값을 가져오고 첫번째 인자를 입력하면 입력된 값으로 countPerPageSet가 지정 됩니다.

인스턴스 내부의 값만 변경 되므로 이후 bind 메서드를 호출 해 줘야 바뀐 부분이 화면에 표시됩니다.

N.pagination 이 이미 인스턴스화 된 후 페이징 수를 변경할때 사용 됩니다.
countPerPageSet number N/A 입력된 값으로 컴포넌트 옵션의 countPerPageSet가 지정 됩니다.
currPageNavInfo N/A N/A N.pagination

페이징 정보 객체를 반환 합니다.

Returns the paging information object.

currPageNavInfo 객체 정보

currPageNavInfo object information

  • pageNo : 현재 페이지 번호.
  • pageNo : Current page numbe.r
  • countPerPage : 페이지 당 행 수
  • countPerPage : number of rows per page.
  • countPerPageSet : 페이지 세트당 페이지 수.
  • countPerPageSet : number of pages per page set.
  • currSelPageSet : 현재 페이지 세트 번호.
  • currSelPageSet : Current page set number.
  • pageCount : 전체 페이지 수.
  • pageCount : Total number of pages.
  • pageSetCount : 전체 페이지 세트 수.
  • pageSetCount : Total number of page sets.
  • totalCount : 전체 행 수.
  • totalCount : Total number of rows.
  • startPage : 현재 페이지 세트 중 첫번째 페이지 번호.
  • startPage : Number of the first page in the current page set.
  • startRowIndex : 선택한 페이지의 첫번째 행 인덱스.
  • startRowIndex : Index of the first row on the selected page.
  • startRowNum : 선택한 페이지의 첫번째 행 번호
  • startRowNum : Number of the first row of the selected page.
  • endPage : 현재 페이지 세트 중 마지막 페이지 번호.
  • endPage : Number of the last page in the current page set.
  • endRowIndex : 선택한 페이지의 마지막 행 인덱스.
  • endRowIndex : Index of the last row on the selected page.
  • endRowNum : 선택한 페이지의 마지막 행 번호.
  • endRowNum : Number of the last row of the selected page.
