Name Arguments Type Return Description
data N/A N/A array[json object]|jquery object 컴포넌트에 바인드 되어 있는 최종 데이터를 반환 합니다.
selFlag boolean N/A 반환되는 데이터의 타입을 결정하는 구분자 입니다.

selFlag 인자 값에 따라 다음과 같은 타입의 데이터를 반환 합니다.

  • 입력하지 않았거나 undefined : array[json object] 타입의 가공 된 데이터를 반환 합니다.

  • false : 바인드 되어 있는 원형(jquery object 타입)의 데이터를 반환 합니다.

    다른 데이터 관련 컴포넌트와 양방향 데이터 바인딩을 가능하게 하려면 반드시 false 인자값을 넣어 줘야 합니다.
context N/A N/A jquery object 컴포넌트를 적용한 요소를 반환 합니다.
selector string N/A selector 문자열을 입력하면 context 옵션으로 지정한 요소에서 선택된 요소를 반환 합니다.
bind N/A N/A N.pagination 지정된 context에 데이터를 바인드 합니다.
data|totalCount string N/A 숫자 타입의 totalCount 값을 넣으면 컴포넌트 인스턴스의 totalCount 옵션이 지정한 값으로 치환되고 array(object) 타입의 data 인자를 지정하면 컴포넌트에 바인드 된 데이터가 지정한 데이터로 치환되어 바인드 됩니다.
totalCount number N/A 페이징 될 전체 로우 카운트가 다시 지정 됩니다.
pageNo N/A N/A number|N.pagination

페이지번호 값을 가져오거나 지정 합니다.

인자가 없으면 pageNo 값을 가져오고 첫번째 인자를 입력하면 입력된 값으로 pageNo가 지정 됩니다.

인스턴스 내부의 값만 변경 되므로 이후 bind 메서드를 호출 해 줘야 바뀐 부분이 화면에 표시됩니다.

N.pagination 이 이미 인스턴스화 된 후 페이지를 변경할때 사용 됩니다.

인스턴스 내부의 값만 변경 되므로 이후 bind 메서드를 호출 해 줘야 바뀐 부분이 화면에 표시됩니다.

N.pagination 이 이미 인스턴스화 된 후 전체 행 수를 변경할때 사용 됩니다.
totalCount number N/A 입력된 값으로 컴포넌트 옵션의 totalCount가 지정 됩니다.
countPerPage N/A N/A number|N.pagination

페이지 당 행 수 값을 가져오거나 지정 합니다.

인자가 없으면 countPerPage 값을 가져오고 첫번째 인자를 입력하면 입력된 값으로 countPerPage가 지정 됩니다.

인스턴스 내부의 값만 변경 되므로 이후 bind 메서드를 호출 해 줘야 바뀐 부분이 화면에 표시됩니다.

N.pagination 이 이미 인스턴스화 된 후 페이지당 행수를 변경할때 사용 됩니다.

인자가 없으면 countPerPageSet 값을 가져오고 첫번째 인자를 입력하면 입력된 값으로 countPerPageSet가 지정 됩니다.

인스턴스 내부의 값만 변경 되므로 이후 bind 메서드를 호출 해 줘야 바뀐 부분이 화면에 표시됩니다.

N.pagination 이 이미 인스턴스화 된 후 페이징 수를 변경할때 사용 됩니다.

currPageNavInfo N/A N/A N.pagination

페이징 정보를 반환 합니다.

currPageNavInfo 정보
  - pageNo : 선택한 페이지 번호
  - countPerPage : 페이지당 행 수
  - countPerPageSet : 페이지 세트당 페이지 수
  - currSelPageSet : 현재 페이지세트(페이지 그룹) 번호
  - pageCount : 전체 페이지 개수
  - pageSetCount : 전체 페이지세트(페이지 그룹) 개수
  - totalCount : 전체 행 개수
  - startPage : 현재 페이지세트(페이지 그룹) 중 첫번째 페이지 번호
  - startRowIndex : 선택한 페이지의 첫번째 행 인덱스
  - startRowNum : 선택한 페이지의 첫번째 행 번호
  - endPage : 현재 페이지세트(페이지 그룹) 중 마지막 페이지 번호
  - endRowIndex : 선택한 페이지의 마지막 행 인덱스
  - endRowNum : 선택한 페이지의 마지막 행 번호
