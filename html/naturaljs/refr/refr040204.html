기본옵션 Default Options

Name Type Default Required Description
context jquery object|jquery selector string null O

N.button을 적용 할 context 요소를 지정 합니다.

Specifies the context element to apply N.button.

N.button의 context 요소는 반드시 a 나 button, input[type=button] 태그로 작성 해야 합니다.

The context element of N.button must be written with a or button, input[type=button] tag.

N.button은 한번에 여러 요소를 선택해서 context 로 지정 할 수 있습니다.

N.button can select multiple elements at once and set it as context.

<a class="button-context" data-opts='{ "color" : "white", size : "big" }'>Button</a>
<input class="button-context" type="button" value="Button" data-opts='{ "color" : "gray" }'>
<button class="button-context" data-opts='{ "color" : "green", size : "large" }'>Button</button>
size string "medium" X

크기

smaller, small, medium, large, big
color string "white" X

색깔

white, blue, skyblue, gray, green, yellowgreen
