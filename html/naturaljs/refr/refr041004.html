기본옵션 Default Options

Name Type Default Required Description
data array[json object] undefined O

N.pagination에 바인딩 할 데이터를 지정 합니다.

Specifies the data to bind to N.pagination.

data가 입력되면 totalCount 값이 자동으로 계산되어 들어가므로 totalCount 값을 0으로 지정하거나 지정하지 않아야 합니다.

서버에서 DB 쿼리로 페이징 시 에는 data 옵션을 지정하지 말고 totalCount 값만 서버에서 얻어와 지정하기 바랍니다.
ex) N().pagination({ totalCount : 1234 });

context jquery object|jquery selector string null O

N.pagination을 적용 할 context 요소를 지정 합니다.

Specifies the context element to apply N.pagination.

N.pagination의 context 요소는 반드시 div 태그 안에 ul, li 태그로 작성 해야 합니다.

The context element of N.pagination must be written with ul, li tag in div tag.

  • 첫 페이지, 마지막 페이지, 이전 페이지, 다음페이지
    • ul 태그의 li 태그에 a 태그를 넣어서 사용 합니다.
    • 첫 페이지, 이전 페이지 요소와 마지막페이지, 다음페이지 요소는 ul 태그를 분리하여 구성 합니다.
    • 첫 페이지, 마지막페이지 요소는 필요하지 않다면 해당 li 태그를 작성하지 않으면 해당 기능이 비활성화 됩니다.
  • 페이지 인덱스
    • ul 태그의 li 태그에 a 태그를 넣어서 사용 합니다.
<div class="pagination-context">
    <ul>
        <li><a href="#">first</a></li>
        <li><a href="#">prev</a></li>
    </ul>
    <ul>
        <li><a href="#"><span>1</span></a></li>
    </ul>
    <ul>
        <li><a href="#">next</a></li>
        <li><a href="#">last</a></li>
    </ul>
</div>
totalCount number 0 O

페이징 될 전체 로우 카운트를 지정 합니다.

서버에서 DB 쿼리로 페이징 시 에는 data 옵션을 지정하지 말고 totalCount 값만 서버에서 얻어와 지정하기 바랍니다.
countPerPage number 10 한번에 몇개의 로우데이터를 반환 할 것인지 지정 합니다.
countPerPageSet number 10 페이징 번호를 몇개를 표시할 것인지 지정 합니다.
pageNo number 1 초기화시 처음 표시 할 페이지 번호를 지정 합니다.
onChange function null 페이지가 전환 될때 발생되는 이벤트 입니다. 
onChange : function(pageNo, selEle, selData, currPageNavInfo) {
    // this 는 N.pagination 인스턴스
    // pageNo 는 선택한 페이지 번호
    // selEle 는 선택한 페이지 링크 요소
    // selData 는 선택한 페이지가 가지고 있는 데이터 리스트
    // currPageNavInfo 는 이벤트가 발생할 시점의 페이징 정보
}

서버에서 DB 쿼리로 페이징 시 먼저 조회하고자 하는 전체 행수를 가져오고 onChange 이벤트 핸들러의 인자로 들어오는 currPageNavInfo 정보로 선택될 DB 로우의 시작값(currPageNavInfo.startRowNum)과 얼마만큼 가져올지 등의 정보(currPageNavInfo.countPerPage)를 서버에 파라미터로 전송한 결과 데이터를 N.grid 에 bind하는 형태로 DB 페이징을 처리 할 수 있습니다.

blockOnChangeWhenBind 옵션이 true 이면 페이징 버튼들을 클릭 할 때만 이벤트가 실행 됩니다.

currPageNavInfo 정보
  - pageNo : 선택한 페이지 번호
  - countPerPage : 페이지당 행 수
  - countPerPageSet : 페이지 세트당 페이지 수
  - currSelPageSet : 현재 페이지세트(페이지 그룹) 번호
  - pageCount : 전체 페이지 개수
  - pageSetCount : 전체 페이지세트(페이지 그룹) 개수
  - totalCount : 전체 행 개수
  - startPage : 현재 페이지세트(페이지 그룹) 중 첫번째 페이지 번호
  - startRowIndex : 선택한 페이지의 첫번째 행 인덱스
  - startRowNum : 선택한 페이지의 첫번째 행 번호
  - endPage : 현재 페이지세트(페이지 그룹) 중 마지막 페이지 번호
  - endRowIndex : 선택한 페이지의 마지막 행 인덱스
  - endRowNum : 선택한 페이지의 마지막 행 번호
blockOnChangeWhenBind boolean false

bind 함수를 호출 할 때 onChange 이벤트를 차단 할지 여부

Whether to block onChange event when call the bind method
