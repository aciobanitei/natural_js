N.element 객체의 함수 Function of N.element Object

"N.element.아래함수명(arg[0...N])" 과 같이 사용하면 됩니다.

Name Arguments Type Return Description
toData N/A N/A boolean

인자로 입력한 입력 요소들의 id 와 value 속성 값으로 JSON 오브젝트를 생성 합니다.

N.form 에서 add 메서드 호출 시 초기 데이터를 생성 할 때 사용 합니다.

// #box 의 입력요소의 값을 json 데이터로 변환
var data = N.element.toData($("#box").find(":input"));
eles jquery object N/A 입력 요소
maxZindex N/A N/A number

입력한 요소 중 가장 높은 z-index 값을 반환 합니다.

var maxZindex = N.element.maxZindex($("div"));
ele jquery object N/A

대상 요소

입력하지 않으면 $("div, span, ul, p, nav, article, section") 로 작동 합니다.
