예제 Examples

1. 지정한 영역에 다른 페이지 불러오면서 불러온 페이지에 데이터 전달 하기

1.1. Communicator 초기 화 후 submit 전에 request.attr("attribute name", object) 명령을 실행.

<section id="section"></section>

<script type="text/javascript">N("#section").comm("page.html").request.attr("data1", { data : ["1", "2"] }).request.attr("data2", ["3", "4"]).submit();
</script>

1.2. 불러온 페이지의 Controller 오브젝트의 init 메서드의 두번째 인자인 request 에서 request.attr("attribute name") 명령으로 전달 된 데이터를 꺼냄.

N("page selector").cont({
	init : function(view, request) {
		var data1 = request.attr("data1");
		var data2 = request.attr("data2");
	}
});
