N.json 객체의 함수 Function of N.json Object

"N.json.아래함수명(arg[0...N])" 과 같이 사용하면 됩니다.

Name Arguments Type Return Description
mergeJsonArray - - -

json object 가 담겨 있는 array 두개를 병합 합니다.

arr1 인자를 기준으로 arr2 를 병합 하고 중복되는 요소가 있으면 해당 요소는 병합하지 않습니다.

세번째 인자로 오브젝트의 프로퍼티명을 지정 하면 해당 프로퍼티를 기준으로 중복된 값이 있다면 해당 오브젝트는 병합 하지 않습니다.

병합할 때 원본 데이터를 확장하는 개념이어서 원본 데이터의 메모리 참조는 변경 되지 않고 유지 됩니다.
arr1 array[json object] - 원본 array[json object]
arr2 array[json object] - 병합 할 array[json object]
key string - 병합 할 기준 프로퍼티명(필수 인자 아님)
format - - - json object 를 보기 좋게 format 해 줍니다.
oData json object - json object
sIndent number - indent, 코드 그룹 간 공백 개수
