예제 Examples

1. 선언형 옵션을 적용하여 버튼생성

<a class="button" data-opts='{ "color" : "white", "size" : "big" }'>a</a>
<input class="button" type="button" value="input button" data-opts='{ "color" : "gray" }'>
<button class="button" data-opts='{ "color" : "green", "size" : "large" }'>button</button>

<script type="text/javascript">
       N(".button").button();
</script>

2. 명령형 옵션을 적용하여 버튼생성

<a class="aTag">a</a>

<script type="text/javascript">
       N(".aTag").button({
           color: "blue",
           size: "small"
       });
</script>
