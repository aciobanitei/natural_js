기본옵션 Default Options

Name Type Default Required Description
context jquery object|jquery selector string null O

N.datepicker가 적용 될 입력 요소를 지정 합니다.

Specifies the input element to which the N.datepicker is applied.

<input class="datepicker-context" type="text">
yearsPanelPosition string(left|top) left

top으로 설정하면 연도 선택 요소가 상단에 생성 됩니다.

If set to top, the year selection element is created at the top.
monthsPanelPosition string(left|top) left

top으로 설정하면 월 선택 요소가 상단에 생성 됩니다.

If set to top, the month selection element is created at the top.
minYear number 200

yearsPanelPosition 옵션 값이 "top" 일 때 선택 할 수 있는 시작 연도를 지정 합니다.

yearsPanelPosition Specifies the starting year that can be selected when the value is "top".
maxYear number 200

yearsPanelPosition 옵션 값이 "top" 일 때 선택 할 수 있는 마지막 연도를 지정 합니다.

yearsPanelPosition Specifies the last year that can be selected when the value is "top".
yearChangeInput boolean false

true로 설정하면 연도 변경 시 변경된 날짜가 입력요소에 바로 반영 됩니다.

If set to true, the changed date is applied immediately to the input element when the year is changed.
monthChangeInput boolean false

true로 설정하면 월 변경 시 변경된 날짜가 입력요소에 바로 반영 됩니다.

If set to true, the changed date is applied immediately to the input element when the month is changed.
touchMonthChange boolean false

true로 설정하면 좌우로 터치 드래그 했을 때 월이 변경 됩니다.

If set to true, the month changes when you touch-drag left or right.
scrollMonthChange boolean false

true로 설정하면 마우스 휠을 스크롤 했을 때 월이 변경 됩니다.

If set to true, the month will change when the mouse wheel is scrolled.
monthonly boolean false

true로 설정하면 연도와 월만 선택 할 수 있는 Monthpicker 가 표시 됩니다.

When set to true, the Monthpicker is displayed, allowing you to select only the year and month.
focusin boolean true

false로 설정하면 입력요소에 커서가 Focus 되었을때 Datepicker가 표시 되지 않습니다.

If set to false, the datepicker is not displayed when the cursor is focused on the input element.
onSelect function null

날짜나 월(monthonly 옵션이 true로 설정 되었을 경우)이 선택 되었을 때 실행되는 이벤트 핸들러를 정의 합니다.

Defines an event handler that is executed when a date or month(if the monthonly option is set to true) is selected.

onSelect : function(context, selDate, monthonly) {
    // this : N.datepicker 인스턴스
    // context : context 요소
    // selDate : 선택 된 날짜의 Date 객체
    // monthonly : monthonly 옵션 값
}
onSelect : function(context, selDate, monthonly) {
    // this : N.datepicker instance
    // context : context element
    // selDate : The Date object of the selected date
    // monthonly : monthonly option value
}

    // this : N.datepicker instance
