함수 Methods

Name Arguments Type Return Description
context N/A N/A jquery object 컴포넌트를 적용한 요소를 반환 합니다.
selector string N/A selector 문자열을 입력하면 context 옵션으로 지정한 요소에서 선택된 요소를 반환 합니다.
open N/A N/A N.popup 팝업을 보여 줍니다.
onOpenData object N/A 불러올 팝업 페이지의 onOpen 이벤트 헨들러 함수의 첫번째 인자에 지정한 onOpenData를 전달 합니다.
close onCloseData object N.popup 팝업을 숨깁니다.
remove N/A N/A N.popup 팝업을 표현하는 HTML Element들을 제거합니다.
