jQuery plugin 확장 메서드 jQuery plugin extension methods

jQuery plugin 형태로 제공 되는 CORE 유틸리티 함수들은 다음과 같습니다.

CORE utility functions provided as jQuery plugin types are as follows.

Name Arguments Type Return Description
N("selector").instance N/A N/A object

UI 컴포넌트의 context 요소나 View 요소에서 컴포넌트 객체의 인스턴스 또는 Controller object를 반환 하거나 객체 인스턴스를 요소에 저장 합니다.

Returns the instance of the component object or the Controller object from the context element or View element of the UI component, or saves the object instance to the element.

Natural-JS 는 탭이나 팝업 등의 블록 컨텐츠를 쉽게 제어하기 위해서 컴포넌트나 라이브러리 초기화 시 지정한 템플릿(context 나 view) 요소에 생성 된 객체 인스턴스를 저장 합니다.

Natural-JS stores the created object instance in the template (context or view) element specified when initializing the component or library to easily control block contents such as tabs or pop-ups.

다음과 같이 인자의 개수와 타입에 따라서 다르게 작동 됩니다.

It works differently depending on the number and type of parameters as follows.

  1. 선택한 요소에 저장되어 있는 모든 인스턴스를 반환.

    Returns all instances stored in the specified element.

    반환 된 인스턴스가 1이면 원래 인스턴스 객체가 반환됩니다. 인스턴스가 두 개 이상인 경우 배열에 저장되어 반환 됩니다. 반환 된 인스턴스가 없으면 undefined가 반환됩니다.

    If the returned instance is 1, the original instance object is returned. If there are more than one instance, they are stored in an array and returned. If no instance is returned, undefined is returned.

    var all = N(".grid01", ".grid02").instance();

  2. 선택한 요소에 저장되어 있는 모든 인스턴스를 콜백의 함수의 인자로 반환.

    Returns all instances stored in the selected element as arguments of the callback function.

    인스턴스의 개수 만큼 콜백함수가 실행 됩니다.

    Callback function is executed as many as the number of instances.

    var all = N(".grid01", ".grid02").instance(function(instanceId, instance) {
    // this : instance
    // instanceId : lang="ko_KR">저장 된 인스턴스의 식별자Identifier of the stored instance
    // instance : 저장 된 인스턴스Stored instance
});

  3. 선택한 요소에 instanceId 가 "name" 으로 저장되어 있는 모든 인스턴스를 반환.

    Returns all instances whose instanceId is stored as "name" in the selected element.

    반환 된 인스턴스가 1이면 원래 인스턴스 객체가 반환됩니다. 인스턴스가 두 개 이상인 경우 배열에 저장되어 반환 됩니다. 반환 된 인스턴스가 없으면 undefined가 반환됩니다.

    If the returned instance is 1, the original instance object is returned. If there are more than one instance, they are stored in an array and returned. If no instance is returned, undefined is returned.

    var all = N(".grid01", ".grid02").instance("name");

  4. 선택한 요소에 instanceId 가 "name" 으로 저장되어 있는 모든 인스턴스를 콜백 함수의 인자로 반환.

    Return all instances where instanceId is stored as "name" in the selected element as argument of callback function.

    인스턴스의 개수 만큼 콜백함수가 실행 됩니다.

    Callback function is executed as many as the number of instances.

    var all = N(".grid01", ".grid02").instance("name", function(instanceId, instance) {
    // this : instance
    // instanceId : lang="ko_KR">저장 된 인스턴스의 식별자Identifier of the stored instance
    // instance : 저장 된 인스턴스Stored instance
});

  5. 선택한 요소에 instance 를 "name" 으로 저장.

    Store instance as "name" in the selected element.

    N(".grid01").instance("name", instance);

    인스턴스 저장 시 instance 인자가 function 타입이면 정상작동 하지 않을 수 있습니다. function 타입을 제외한 object 나 string 과 같은 타입을 instance 인자로 지정 해야 합니다.

    When saving an instance, if the instance argument is a function type, it may not work properly. Types such as object or string, except the function type, should be specified as instance arguments.
name string N/A

인스턴스 명

Instance name

  • Controller object of N.cont : cont

  • N.alert 의 instance : alert

    Instance of N.alert : alert

    .block_overlay_msg__ 요소에 저장 되어 있음.

  • Instance of N.button : button
  • N.datepicker 의 instance : datepicker
  • Instance of N.datepicker : datepicker

    The controller object of the loaded pop-up content is stored in the .block_overlay_msg__ > .msg_box__ > .view_context__ element.

    Instance of N.tab : tab

    로드 된 탭 컨텐츠의 Controller object 는 .tab__ > .{tab 컨텐츠 요소 id} > .view_context__ 요소에 저장 되어 있음.

    The controller object of the loaded tab content is stored in the .tab__ > .{tab 컨텐츠 요소 id} > .view_context__ element.

  • Instance of N.select : select

  • Instance of N.list : list

인스턴스의 저장 위치가 별도로 명시되어 있지 않은 컴포넌트는 context 옵션으로 지정한 요소에 저장 됩니다.

Components that do not specify where to store the instance are stored in the element specified by the context option.
instance object|function N/A

선택한 요소에 저장할 인스턴스나 인스턴스를 가져올 callback function.

Callback function to get the instance or instance to be saved in the selected element.

callback의 인자로 index(arguments[0])와 각각의 instance(arguments[1])가 반환 됩니다. 이 callback function 의 "this" 는 각각의 instance를 가리킵니다.

As the argument of callback, index(arguments[0]) and each instance(arguments[1]) are returned. This callback function's "this" points to each instance.
N("selector").tpBind N/A N/A jquery object

선택한 요소에 같은 이름의 이벤트 정의 되어 있더라도 tpBind 로 정의한 이벤트가 제일 먼저 실행 되게 해 줍니다.

Even if an event of the same name is defined in the selected element, the event defined by tpBind is executed first.

기본적인 사용법은 jQuery().bind 메서드와 같습니다.

The basic usage is the same as the jQuery().bind method.
event string N/A

N("selector").vals N/A N/A string|array|jquery object

select, select[multiple=multiple], input[type=radio], input[type=checkbox] 와 같은 선택 요소의 선택된 값을 가져오거나 선택 합니다.

Gets or selects the selected value of a selection element such as select, select[multiple=multiple], input[type=radio], input[type=checkbox].

  • vals 인자를 입력하지 않으면 선택한 값이 반환됩니다. vals 인자가 지정되면 입력 한 값과 일치하는 선택 요소가 선택됩니다.
  • If the vals argument is not entered, the selected values ​​are returned. If the vals argument is specified, option elements that match the entered values ​​are selected.
  • 하나만 선택하면 string 유형의 값이 하나만 반환되고 둘 이상인 경우 값이 배열로 반환됩니다.
  • If only one is selected, only one value of type string is returned, and if it is more than two, values ​​are returned in an array.

checkbox 의 경우 선택지가 1개만 있다면 Y/N 또는 1/0 과 같이 여부를 결정하는 모드로 작동하게 됩니다.

In the case of checkbox, if there is only one option, it works in decision mode like Y/N or 1/0.

단일 선택의 표준 값이 Y/N 인지 1/0 인지 on/off 인지 설정은 natural.config.js 의 N.context.attr("core").sgChkdVal("체크값") 변수와 N.context.attr("core").sgUnChkdVal("언체크값") 변수 값으로 설정 할 수 있습니다.

Whether the standard value of a single selection is Y / N, 1/0, or on / off can be set to the N.context.attr("core").sgChkdVal("check value") and N.context.attr("core").sgChkdVal("uncheck value") variable values ​​of natural.config.js.
vals array|string|function N/A

단일 값을 지정 할 때는 문자열로 값을 지정하고 둘 이상의 선택지를 선택 할 때는 배열에 문자열을 담아 값을 지정 합니다.

When specifying a single value, specify the value as a string and when selecting two or more options, specify the value by storing the string in an array.

function 지정하면 선택 된 선택지 요소 개수 만큼 지정한 콜백 함수를 실행 해 줍니다. 콜백함수의 인자는 다음과 같습니다.

If function is specified, the specified callback function is executed as many as the number of selected option elements. The arguments of the callback function are as follows.

  • this : 선택 된 선택지 요소
  • this : Selected option element
  • arguments[0] : 선택 된 선택지 요소의 index
  • arguments[0] : Index of selected option element
  • arguments[1] : 선택 된 선택지 요소
  • arguments[1] : Selected option element

N(selector).events N/A N/A object similar to array|array|undefined

선택한 요소에 바인드 되어 있는 이벤트들을 반환 합니다.

Returns the events bound to the selected element.

eventName 인자와 namespace 인자를 둘다 입력 하지 않으면 모든 이벤트들을 반환하고 namespace 를 입력 하지 않으면 지정한 eventName 에 해당하는 이벤트들만 반환 합니다.

If both the eventName and namespace arguments are not entered, all events are returned. If no namespace is entered, only the events corresponding to the specified eventName are returned.

namespace 인자가 입력 되면 배열 객체 타입으로 이벤트를 반환 하고 그렇지 않은 경우에는 jquery object 타입으로 이벤트들을 반환 합니다. 바인드되어 있는 이벤트가 없으면 undefined 를 반환 합니다.

If the namespace argument is entered, the event is returned as an array object type. Otherwise, the event is returned as a jquery object type. If there are no bound events, undefined is returned.
eventName string N/A
