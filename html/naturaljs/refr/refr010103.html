jQuery plugin 확장 메서드 jQuery plugin extension methods

jQuery plugin 타입으로 제공 되는 CORE 유틸리티 함수들은 다음과 같습니다.

Name Arguments Type Return Description
N("selector").instance N/A N/A object

UI 컴포넌트의 context 요소에서 컴포넌트 객체의 인스턴스 또는 View 요소에 저장 된 Controller object를 반환 하거나 객체 인스턴스를 요소에 저장 합니다.

Natural-JS 는 탭이나 팝업 등의 블록 컨텐츠를 쉽게 제어하기 위해서 컴포넌트나 라이브러리 초기화 시 지정 한 템플릿(context 나 view) 요소에 생성 된 객체 인스턴스를 저장 합니다.

다음과 같이 인자의 개수와 타입에 따라서 다르게 작동 됩니다.

  1. 조회한 요소에 저장되어 있는 모든 인스턴스를 array 타입으로 모두 반환. 
    var all = N("div#sample1", "div#sample2").instance();
  2. 조회한 요소에 저장되어 있는 모든 요소를 지정한 콜백의 인자로 반환. 
    var all = N("div#sample1", "div#sample2").instance(function(instanceId, instance) {
    // this : instance
    // instanceId : 저장 된 인스턴스의 식별자
    // instance : 저장 된 인스턴스
});
  3. 조회한 요소에 instanceId 가 "name" 으로 저장되어 있는 모든 요소를 반환. 
    // 반환된 인스턴스가 1개 이면 인스턴스 원형을 반환하고
// 1개 이상이면 array 에 담아 반환 하고
// 반환된 인스턴스가 없다면 undefined 를 반환 합니다.
var all = N("div#sample1", "div#sample2").instance("name");
  4. 조회한 요소에 instanceId 가 "name" 으로 저장되어 있는 모든 요소를 지정한 콜백의 인자로 반환. 
    var all = N("div#sample1", "div#sample2").instance("name", function(instanceId, instance) {
    // this : instance
    // instanceId : 저장 된 인스턴스의 식별자
    // instance : 저장 된 인스턴스
});
  5. 조회한 요소에 instanceId 를 "name" 으로 정의 해서 instance 를 저장. 
    N("div#sample1").instance("name", instance);

    인스턴스 저장 시 instance 인자가 function 타입이면 정상작동 하지 않을 수 있습니다. function 타입을 제외한 object 나 string 과 같은 타입을 instance 인자로 지정하기 바랍니다.
name string N/A

인스턴스 명

미리 정의 되어 있는 인스턴스 명은 다음과 같습니다.

  • N.cont 의 Controller object : cont

  • N.alert 의 instance : alert

    .block_overlay_msg__ 요소에 저장 되어 있음.

  • N.button 의 instance : button
  • N.datepicker 의 instance : datepicker

  • N.popup 의 instance : popup

    로딩 된 팝업 컨텐츠의 Controller object 는 .block_overlay_msg__ > .msg_box__ > .view_context__ 요소에 저장 되어 있음.

  • N.tab 의 instance : tab

    로딩 된 탭 컨텐츠의 Controller object 는 .tab__ > .{tab 컨텐츠 요소 id} > .view_context__ 요소에 저장 되어 있음.

  • N.select 의 instance : select
  • N.form 의 instance : form
  • N.list 의 instance : list
  • N.grid 의 instance : grid
  • N.pagination 의 instance : pagination
  • N.tree 의 instance : tree
  • N.notify 의 instance : notify
  • N.docs 의 instance : docs

인스턴스의 저장 위치가 별도로 명시되어 있지 않은 컴포넌트는 context 옵션으로 지정한 요소에 저장 됩니다.
instance object|jquery object N/A

선택 한 요소에 저장 할 인스턴스나 인스턴스를 가져올 callback function

callback의 인자로 index(arguments[0])와 각각의 instance(arguments[1])가 반환 됩니다. 이 callback function 의 this 는 각각의 instance를 가리킵니다.
N("selector").tpBind N/A N/A jquery object

선택 한 요소에 지정 한 이벤트 명과 같은 이벤트가 미리 정의 되어 있더라도 정의한 이벤트가 제일 먼저 실행 되게 해 줍니다.

기본적인 사용법은 jQuery().bind 메서드와 같습니다.
event string N/A 이벤트 명
handler function N/A 이벤트 핸들러 함수
N("selector").vals N/A N/A string|array|jquery object

select, select[multiple=multiple], input[type=radio], input[type=checkbox] 와 같은 선택 요소의 선택된 값을 가져오거나 명령으로 선택 할 수 있게 해 주는 함수 입니다.

  • vals 인자를 지정하지 않으면 선택된 value 들을 가져오고 vals를 지정하면 입력한 값들과 일치하는 선택지 요소들을 선택 해 줍니다.
  • 선택 된 값을 가져올 때 1개만 선택 되어 있으면 string 타입의 값 1개만 반환하고 2개 이상 일 경우 값 들을 array 에 담아 반환 합니다.
  • checkbox 의 경우 선택지가 1개만 있다면 Y나 N 또는 1, 0 과 같이 여부를 판단하는 모드로 작동하게 됩니다. Y/N 인지 1/0 인지 on/off 인지 설정은 natural.config.js 의 N.context.attr("core").sgChkdVal(체크값) 변수와 N.context.attr("core").sgUnChkdVal(언체크값) 변수 값으로 설정 할 수 있습니다.
vals array|string|function N/A 단일 값을 지정 할 때는 string 타입으로 값을 지정하고 2개 이상의 선택지를 선택 할 때는 array 에 string 을 담아 지정 합니다. function 타입을 지정하면 요소 개수 만큼 지정한 콜백 함수를 실행 해 줍니다. 콜백함수의 인자는 다음과 같습니다.
  • this : 선택 된 선택지 요소
  • arguments[0] : 선택 된 선택지 요소의 index
  • arguments[1] : 선택 된 선택지 요소
N(array).remove_ N/A N/A jquery object 셀럭터로 지정한 array 의 특정 요소를 제거 합니다.
idx number N/A 배열 에서 제거 할 요소의 Index
length number N/A 제거 할 배열 요소들의 개수
N(selector).events N/A N/A object similar to array|array|undefined

셀렉터로 지정한 요소에 바인드 되어 있는 이벤트들을 반환 합니다.

eventName 인자와 namespace 인자를 둘다 입력 하지 않으면 모든 이벤트들을 반환하고 namespace 를 입력 하지 않으면 지정한 eventName 에 해당하는 이벤트들만 반환 합니다.

namespace 인자가 입력 되면 array 타입으로 이벤트를 반환 하고 그렇지 않은 경우에는 array 와 비슷한 object 타입으로 이벤트들을 반환 합니다. 바인드되어 있는 이벤트가 없으면 undefined 를 반환 합니다.
eventName string N/A 이벤트 명
namespace string N/A 이벤트 네임스페이스
