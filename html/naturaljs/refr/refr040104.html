Name Type Default Required Description
context window|jquery object|jquery selector string null O

Alert 의 배경이 되는 컨텐츠를 가려주는 overlay 가 적용될 부분을 jQuery selector 를 사용하여 지정 합니다.

  • window 객체를 지정하면 화면 전체를 덮어 주고 jquery object를 입력하면 지정한 요소만 덮어줍니다.
  • 입력요소(select, input, textarea 등)를 지정 하면 입력요소 옆에 툴팁으로 메시지를 표시 해 줍니다.
msg (jquery object, HTML Tag string)|object undefined O

Alert 로 표시할 메시지

jquery object, HTML 문자열 이나 HTML 요소 모두 인식 합니다.
vars array undefined X

메시지의 변수를 vars로 지정한 값으로 치환 해줍니다.

N(window).alert({
    msg : "{0} 저는 {1}입니다.",
    vars : ["안녕하세요", "홍길동"]
}).show();

/*
결과 메시지 : "안녕하세요 저는 홍길동 입니다."
*/

If set to true, the HTML of the message is applied.

true 이면 [확인], [취소] 버튼을 표시하고 false 이면 [확인] 버튼만 표시 합니다.

width number|function 0 X

메시지 대화상자의 넓이

  • number 타입 일 경우 지정한 수치만큼(px) 넓이가 지정 됩니다.
  • function 일 경우 msgContext(modal 옵션이 true 일때 바닥에 깔리는 요소), msgContents(메시지 컨텐츠 요소) 가 인자로 전달 되고 return 된 값으로 넓이가 지정 됩니다. 
    ...
// 대화상자의 넓이를 화면에 꽉 채우기
width : function(msgContext, msgContents) {
    return $(window).width();
},
...
height number|function 0 X

메시지 대화상자에서 타이틀 영역을 제외한 컨텐츠의 높이

  • number 타입 일 경우 지정한 수치만큼(px) 높이가 지정 됩니다.
  • function 일 경우 msgContext(modal 옵션이 true 일때 바닥에 깔리는 요소), msgContents(메시지 컨텐츠 요소) 가 인자로 전달 되고 return 된 값으로 높이가 지정 됩니다. 
    ...
// 대화상자의 높이를 화면에 꽉 채우기
height: function(msgContext, msgContents) {
    // 팝업이 열릴때 msgContents 는 감춰진 상태여서 show() 함수를 호출 한다음 타이틀 영역의 높이를 가져옴.
    return $(window).height() - msgContents.show().find(".msg_title_box__").height();
},
...
title string undefined X

메시지 대화상자의 제목을 설정 합니다. 설정하지 않으면 타이틀바가 생성 되지 않습니다.

메시지 대화상자의 제목은 context 요소의 title 속성으로도 설정 할 수 있습니다.
button boolean true X

false로 설정 하면 기본 버튼(확인/취소 버튼) 관련 요소들을 생성하지 않습니다.

If set to false, It does not create basic button(OK/Cancel buttons) related elements.
closeMode string remove X

메시지 대화상자가 닫혔을 때 대화상자 요소에 대한 처리 방법을 지정 합니다.

  • hide : 메시지 대화상자 요소를 숨겨 이전 상태를 유지 합니다.
  • remove : 메시지 대화상자 요소를 제거하여 상태를 초기화 합니다.

If set to true, The message dialog is always displayed at the top.
alwaysOnTopCalcTarget string "div, span, ul, p" X

alwaysOnTop 옵션 적용 시 최 상위 z-index를 계산하기 위한 대상 요소들을 지정 합니다.

When applying the alwaysOnTop option, specify the target elements for calculating the highest z-index.

jQuery selector 구문으로 지정 합니다.

It is specified using jQuery selector syntax.

N.alert 이나 N.popup 관련 요소들이 다른 요소에 의해 가려 질 때 가리고 있는 요소 셀렉터를 추가 해 주세요.

Add an element selector that is obscured when elements related to N.alert or N.popup are obscured by other elements.
dynPos boolean true X

false로 설정하면 브라우저의 크기를 조절 하거나 부모 컨텐츠의 높이가 동적으로 변경 되었을때 블록 오버레이의 크기와 메시지 대화상자의 위치를 자동으로 조정 되지 않습니다.

If set to false, It does not automatically adjust the size of the block overlay and the location of the message dialog when the browser is resized or the height of the parent content is dynamically changed.
draggable boolean false X

true로 설정하면 제목 표시 줄로 메시지 대화 상자를 드래그 할 수 있습니다.

If set to true, the message dialog will be draggable by the title bar.

false로 설정하면 메시지 대화상자를 화면 바깥으로 드롭 했을 때 자동으로 내부로 이동 시키지 않습니다.

If set to false, the message dialog will not automatically move inside when dropped off the screen.

Specifies the position to move to inside when the message dialog is dropped off the screen.

메시지 대화상자가 내부로 완전하게 돌아오지 않아 화면에 스크롤바가 생겼을 때 1씩 증감하여 메시지 대화상자의 위치를 보정 하세요.

Correct the position of the message dialog by incrementing or decrementing by 1 when a scroll bar appears on the screen because the message dialog does not completely return to the inside.

true로 설정 하면 메시지 대화상자 요소 위에서 마우스 휠로 스크롤 할 때 브라우저의 윈도우 스크롤을 비활성화 합니다.

If set to true, disables browser window scrolling when scrolling with the mouse wheel over message dialog elements.

메시지 대화상자 요소가 처음이나 마지막으로 스크롤 되었을 때 브라우저 윈도우 스크롤이 위나 아래로 스크롤되는 브라우저의 기본 동작을 차단 합니다.

Blocks the browser's default behavior of scrolling the browser window up or down when the message dialog element is scrolled first or last.
