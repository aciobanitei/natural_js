기본옵션

Default Options

Name Type Default Required Description
context window|jquery object null O

Alert 의 배경이 되는 컨텐츠를 가려주는 overlay 가 적용될 부분을 jQuery selector 를 사용하여 지정 합니다.

window 객체를 지정하면 화면 전체를 덮어 주고 jquery object를 입력하면 지정한 요소만 덮어줍니다.

입력요소(select, input, textarea 등)를 지정 하면 입력요소 옆에 툴팁으로 메시지를 표시 해 줍니다.
msg (jquery object, HTML Tag string)|object undefined O

Alert 로 표시할 메시지

jquery object, HTML 문자열 이나 HTML 요소 모두 인식 합니다.
vars array undefined X

메시지의 변수를 vars로 지정한 값으로 치환 해줍니다.

N(window).alert({
	msg : "{0} 저는 {1}입니다.",
	vars : ["안녕하세요", "홍길동"]
}).show();

/*
결과 메시지 : "안녕하세요 저는 홍길동 입니다."
*/
html boolean false X 메시지에 HTML 을 허용 할지 여부
confirm boolean false X

true 이면 [확인], [취소] 버튼을 표시하고 false 이면 [확인] 버튼만 표시 합니다.
top number undefined X 메시지 대화상자의 상단(px) 위치
left number undefined X 메시지 대화상자의 좌측(px) 위치
width number|function 0 X 메시지 대화상자의 넓이(px)
height number|function 0 X 메시지 대화상자에서 타이틀 영역을 제외한 컨텐츠의 높이(px)
title string undefined X

메시지 대화상자의 제목

메시지 박스(context) 요소에 title 속성값이 정의 되어 있으면 정으된 값으로 지정되며 title 옵션이 undefined 이면 타이틀바를 표시하지 않습니다.
button boolean true X 버튼 표시 여부
closeMode string remove X X 버튼이나 확인 버튼을 클릭 했을때 hide 메서드를 호출 할 것인지 remove 메서드를 호출 할 것인지 지정
- hide : hide 메서드를 호출하여 메시지 대화상자 요소를 숨겨 줌.
- remove : remove 메서드를 호출하여 메시지 대화상자 요소를 제거 함(이전 상태가 사라짐).
modal boolean true X 모달 적용 여부
alwaysOnTop boolean false X 메시지 대화상자를 항상 최 상위에 표시 할건 지 여부
alwaysOnTopCalcTarget string "div, span, ul, p" X alwaysOnTop 옵션 적용 시 최 상위 z-index를 계산하기 위한 대상 요소지정

jQuery selector 문자열로 지정 합니다
dynPos boolean true X 브라우저 창 크기 조절 시, 또는 부모 컨텐츠 높이가 동적으로 변경되었을때 블록 오버레이의 크기와 메시지 대화상자의 위치를 자동으로 조절 할지 여부
draggable boolean false X 메시지 창 드래그 가능 여부
draggableOverflowCorrection boolean true X 메시지 창 드래그 시 메시지 창이 화면바깥으로 넘어갔을때 자동으로 다시 안으로 넣어줄지 여부
draggableOverflowCorrectionAddValues object {
top : 0,
bottom : 0,
left : 0,
right : 0
} 		X 드래그하면서 대화상자가 window영역을 벗어날때 다시 돌아올 위치를 추가로 지정(드래그 시 횡 스크롤이 생겨 화면이 지저분 해 질때 1씩 증감하여 조절 바랍니다.)
overlayClose boolean true X 오버레이 요소를 클릭 했을 때 메세지 대화상자를 닫을지 여부
escClose boolean true X "ESC" 키를 눌렀을 때 메세지 대화상자를 닫을지 여부
windowScrollLock boolean true X modal 옵션이 true 일때 메시지 오버레이 위 에서 스크롤 했을때 윈도우 스크롤을 차단 할 것인지 여부
saveMemory boolean false X 필요없는 참조 요소를 제거하여 메모리를 절약 할 것인지 여부
onOk function null X

확인버튼을 눌렀을때 실행되는 이벤트

onOk : function(msgContext, msgContents) {
	// this : N.alert 인스턴스
	// msgContext : 메시지 오버레이 요소
	// msgContents : 메시지 대화상자 요소
}

이벤트 핸들러 함수 에서 0을 반환 하면 대화 상자를 닫지 않습니다.
onCancel function null X

취소 버튼을 누르거나 X 버튼을 눌렀을때 또는 메시지 오버레이를 클릭하여 메시지 대화상자가 닫혔을때 실행되는 이벤트

onCancel : function(msgContext, msgContents) {
	// this : N.alert 인스턴스
	// msgContext : 메시지 오버레이 요소
	// msgContents : 메시지 대화상자 요소
}

이벤트 핸들러 함수 에서 0을 반환 하면 대화 상자를 닫지 않습니다.
onShow function null X

메시지 대화상자가 표시 된 후 실행되는 이벤트

onShow : function(msgContext, msgContents) {
	// this : N.alert 인스턴스
	// msgContext : 메시지 오버레이 요소
	// msgContents : 메시지 대화상자 요소
}

}
