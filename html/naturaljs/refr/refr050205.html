함수 Methods

Name Arguments Type Return Description
context N/A N/A jquery object

context 요소를 반환 합니다.

Returns the context element.
selector string N/A

jQuery selector 구문을 입력하면 context 요소에서 지정한 요소를 찾아서 반환 합니다.

If you enter jQuery selector syntax, the element specified in the context element is found and returned.
add N/A N/A N.docs 새로운 컨텐츠과 컨텐츠 탭을 추가 합니다.
docId string N/A

document 아이디를 입력 합니다.

이미 같은 아이디로 지정된 컨텐츠가 있다면 해당 컨텐츠가 활성화 됩니다.
docNm string N/A document 이름을 입력 합니다.
docOpts object N/A

추가 되는 페이지의 옵션을 입력 합니다.

var docOpts = {
    url : "url", // document URL
    urlSync : true, // 서버로 요청 할 때의 location.href 와 서버에서 응답이 왔을 때의 location.href 이 같은지 비교하여 다르면 응답을 차단 합니다.
    docId : docId, // document 아이디
    docNm : docNm, // document 명
    onBeforeLoad : function(docId, target) {
        // 추가 되는 document 에만 적용되는 onBeforeLoad 이벤트.
    },
    onLoad : function(docId) {
        // 추가 되는 document 에만 적용되는 onLoad 이벤트.
    },
    onBeforeActive : function(docId) {
        // 추가 되는 document 에만 적용되는 onBeforeActive 이벤트.
    },
    onActive : function(docId) {
        // 추가 되는 document 에만 적용되는 onActive 이벤트.
    },
    onBeforeInactive : function(docId) {
        // 추가 되는 document 에만 적용되는 onBeforeInactive 이벤트.
    },
    onInactive : function(docId) {
        // 추가 되는 document 에만 적용되는 onInactive 이벤트.
    },
    onBeforeRemoveState : function(docId) {
        // 추가 되는 document 에만 적용되는 onBeforeRemoveState 이벤트.
    },
    onRemoveState : function(docId) {
        // 추가 되는 document 에만 적용되는 onRemoveState 이벤트.
    },
    onBeforeRemove : function(docId) {
        // 추가 되는 document 에만 적용되는 onBeforeRemove 이벤트.
    },
    onRemove : function(docId) {
        // 추가 되는 document 에만 적용되는 onRemove 이벤트.
    },
    stateless : false // 상태유지 여부
}

개별 이벤트들은 전역 이벤트가 실행 된 후 실행 됩니다.
active N/A N/A N.docs 선택한 컨텐츠과 컨텐츠 탭을 활성화 합니다.
docId string N/A document 아이디를 입력 합니다.
removeState N/A N/A N.docs 선택한 컨텐츠의 상태를 제거 합니다.
docId string N/A 컨텐츠의 상태를 제거할 document 아이디를 입력 합니다.
remove N/A N/A N.docs 선택한 컨텐츠과 컨텐츠 탭을 제거 합니다.
docId string N/A document 아이디를 입력 합니다.
unconditional boolean N/A true 를 입력 하면 수정 중인 입력이 있더라도 무조건 제거 합니다.
doc N/A N/A object|array

선택한 컨텐츠의 정보(docOpts)를 가져 옵니다.
docId string N/A

정보를 가져 올 document 아이디를 입력 합니다.

입력하지 않으면 모든 컨텐츠의 정보를 array 로 반환 합니다.
cont N/A N/A object

선택한 컨텐츠의 Controller(N.cont) 오브젝트를 가져 옵니다.
docId string N/A

Controller를 가져 올 document 아이디를 입력 합니다.
reload N/A N/A N.docs

선택한 document를 리로드 합니다.

reload 함수 호출 전에 request.attr 함수로 값을 지정하지 않았다면 리로드 전의 request 객체의 값은 리로드 해도 유지 됩니다.
docId string N/A

리로드 할 document 아이디를 입력 합니다.
callback string N/A

리로드가 완료 되면 실행 되는 콜백을 지정 합니다. 필요 없으면 지정 하지 않아도 됩니다.

콜백의 인자로 html 과 Communicator.request 객체가 반환 됩니다.

docsInstance.reload("docId", function(html, request) {
    N.log(html, request);
});
