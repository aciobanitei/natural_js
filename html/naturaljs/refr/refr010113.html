disable N/A N/A boolean

지정한 요소에 바인드 되어 있는 이벤트들의 실행을 중단 시킬 수 있는 이벤트 핸들러 입니다. N().tpBind 메서드를 사용하여 이벤트 중 제일 앞단에 disable 이벤트 핸들러를 바인드 합니다.

// 요소에 클릭 방지 처리
$("selector").tpBind("click", N.event.disable);
// 클릭 방지 처리 해제
$("selector").unbind("click", N.event.disable);
