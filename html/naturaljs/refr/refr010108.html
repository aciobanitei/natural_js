N.date 객체의 함수 Function of N.date Object

"N.date.아래함수명(arg[0...N])" 과 같이 사용하면 됩니다.

Name Arguments Type Return Description
diff N/A N/A string

입력 한 두 날짜 간 일수 차이를 반환 합니다.
refDateStr string N/A 기준 날짜 문자열(예) "19991231")
targetDateStr string N/A 차수를 구할 대상 날짜 문자열(예) "19991231")
strToDateStrArr N/A N/A array

입력 된 날짜 문자열을 입력된 양식을 기준으로 array 타입으로 [년(number), 월(number), 일(number)] 로 반환 합니다.
str string N/A 날짜 문자열
format string N/A 입력한 날짜 문자열의 포멧(필수 입력)
  • Y : 년
  • m : 월
  • d : 일
  • 예) "19991231" : "Ymd"
  • 예) "3112" : "dm"
  • 예) "12311999" : "mdY"
isString string N/A 년월일을 결과 array 에 number 타입 으로 담을지 string 타입으로 담을 지 여부(입력하지 않거나 false 이면 number 타입으로 담음)
strToDate N/A N/A string

입력 된 날짜/시간 문자열을 date object 로 변환하여 객체로 반환 합니다. 반환되는 객체 정보는 다음과 같습니다.

{
    obj : date object,
    format : 날짜 format
}
str string N/A 날짜/시간 문자열
  • "19991231" : 1999년 12월 31일 00시 00분 00초 로 변환
  • "1999123103" : 1999년 12월 31일 03시 00분 00초 로 변환
  • "199912310348" : 1999년 12월 31일 03시 48분 00초 로 변환
  • "19991231034856" : 1999년 12월 31일 03시 48분 56초 로 변환
format string N/A

입력한 날짜/시간 문자열의 포멧

  • Y : 년
  • m : 월
  • d : 일
  • H : 시
  • i : 분
  • s : 초
  • 예) "19991231" : "Ymd"
  • 예) "3112" : "dm"
  • 예) "12311999" : "mdY"
  • 예) "19991231120159" : "YmdHis"

format 인자를 입력하지 않으면 입력한 날짜 문자열의 길이에 따라 다음과 같이 자동으로 설정 됩니다.

  • 4자리 : "Y"
  • 6자리 : "Y-m"
  • 8자리 : "Y-m-d"
  • 10자리 : "Y-m-d H"
  • 12자리 : "Y-m-d H:i"
  • 14자리 : "Y-m-d H:i:s"

대쉬(-), 콜론(:) 등의 날짜와 시간을 구분하는 문자는 Config(N.config, natural.config.js 파일) 의 N.context.attr("data").formatter.date 오브젝트의 함수들로 정의 되어 있습니다. 날짜/시간 구분 문자는 해당 위치에서 변경 가능 합니다.
format N/A N/A array

입력 된 날짜/시간 문자열을 입력된 양식을 기준으로 포멧 합니다.
str string N/A 날짜/시간 문자열
format string N/A 입력한 날짜/시간 문자열의 포멧(필수 입력)
  • Y : 년
  • m : 월
  • d : 일
  • H : 시
  • i : 분
  • s : 초
  • 예) N.date.format("19991231120159", "Y-m-d H:i:s") -> "1999-12-31 12:01:59"
  • 예) N.date.format("19991231120159", "Y/m/d H:i:s") -> "1999/12/31 12:01:59"
  • 예) N.date.format("19991231120159", "Y.m.d H:i") -> "1999.12.31 12:01"
  • 예) N.date.format("19991231120159", N.context.attr("data").formatter.date.YmdHis()) -> "1999-12-31 12:01:59"

    N.context.attr("data").formatter.date 오브젝트는 Config(N.config, natural.config.js 파일) 에 정의 되어 있습니다. 날짜/시간 구분 문자는 해당 위치에서 변경 가능 합니다.
dateToTs N/A N/A number

입력 된 date 객체를 timestamp number 로 바꿔 줍니다.

dateObj 인자를 입력하지 않으면 현재 날짜와 시간으로 전환 합니다.
dateObj date object N/A timestamp number 로 전환 할 date object
tsToDate N/A N/A date object

입력 된 timestamp number 를 date 객체로 바꿔 줍니다.

tsNum 인자를 입력하지 않으면 현재 날짜와 시간으로 전환 합니다.
tsNum number N/A date object 로 전환 할 timestamp number
dateList N/A N/A array[array[date]]

월별 날짜 객체 목록을 가져온다.

Get a list of monthly date objects
dateObj date object N/A timestamp number 로 전환 할 date object
tsToDate N/A N/A date object

입력 된 timestamp number 를 date 객체로 바꿔 줍니다.

tsNum 인자를 입력하지 않으면 현재 날짜와 시간으로 전환 합니다.
tsNum number N/A date object 로 전환 할 timestamp number
