N.date 객체의 함수 Function of N.date Object

N.date 은 날짜 제어 관련 유틸리티 함수들을 모아놓은 객체 입니다.

N.gc is a collection object of utility functions related to date control.

"N.date.{함수명}(arg[0...N])" 와 같이 사용 할 수 있습니다.

You can use like "N.date.{function name}(arg[0...N])"

Name Arguments Type Return Description
diff N/A N/A string

입력 한 두 날짜 간 일수 차이를 반환 합니다.

Returns the difference in days between two input dates.

기준 날짜를 8자리 날짜 문자열(YYYYmmdd)로 입력 합니다.

strToDate N/A N/A string

입력 된 날짜/시간 문자열을 date object 로 변환하여 객체로 반환 합니다. 반환되는 객체 정보는 다음과 같습니다.

{
    obj : date object,
    format : 날짜 format
}
str string N/A 날짜/시간 문자열
  • "19991231" : 1999년 12월 31일 00시 00분 00초 로 변환
  • "1999123103" : 1999년 12월 31일 03시 00분 00초 로 변환
  • "199912310348" : 1999년 12월 31일 03시 48분 00초 로 변환
  • "19991231034856" : 1999년 12월 31일 03시 48분 56초 로 변환
format string N/A

입력한 날짜/시간 문자열의 포멧

  • Y : 년
  • m : 월
  • d : 일
  • H : 시
  • i : 분
  • s : 초
  • 예) "19991231" : "Ymd"
  • 예) "3112" : "dm"
  • 예) "12311999" : "mdY"
  • 예) "19991231120159" : "YmdHis"

format 인자를 입력하지 않으면 입력한 날짜 문자열의 길이에 따라 다음과 같이 자동으로 설정 됩니다.

  • 4자리 : "Y"
  • 6자리 : "Y-m"
  • 8자리 : "Y-m-d"
  • 10자리 : "Y-m-d H"
  • 12자리 : "Y-m-d H:i"
  • 14자리 : "Y-m-d H:i:s"

대쉬(-), 콜론(:) 등의 날짜와 시간을 구분하는 문자는 Config(natural.config.js) 의 N.context.attr("data").formatter.date 오브젝트의 함수들로 정의 되어 있습니다. 날짜/시간 구분 문자는 해당 위치에서 변경 가능 합니다.
format N/A N/A array

입력 된 날짜/시간 문자열을 입력된 서식을 기준으로 포멧 합니다.
str string N/A 날짜/시간 문자열
format string N/A 입력한 날짜/시간 문자열의 포멧(필수 입력)
  • Y : 년
  • m : 월
  • d : 일
  • H : 시
  • i : 분
  • s : 초
N.date.format("19991231120159", "Y-m-d H:i:s") // "1999-12-31 12:01:59"
N.date.format("19991231120159", "Y/m/d H:i:s") // "1999/12/31 12:01:59"
N.date.format("19991231120159", "Y.m.d H:i") // "1999.12.31 12:01"
N.date.format("19991231120159", N.context.attr("data").formatter.date.YmdHis()) // "1999-12-31 12:01:59"

N.context.attr("data").formatter.date 오브젝트는 Config(natural.config.js) 에 정의 되어 있습니다. 날짜/시간 구분 문자는 해당 위치에서 변경 가능 합니다.
