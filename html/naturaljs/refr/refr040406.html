예제 Examples

1. 지정한 영역(.popupArea)을 팝업으로 띄우기

<div title="Popup Example" class="popupArea" style="height: 200px;border: 3px dotted #4CAF50;">이 블록이 팝업으로 보이나요?</div>

<script type="text/javascript">
   	N(".popupArea").popup().open();
</script>

2. 다른 (블록)페이지를 팝업으로 띄우기

<script type="text/javascript">
	var popup = N().popup("html/indx/header.html");
	popup.open();
</script>

3. 다른 (블록)페이지를 드래그 가능한 팝업으로 띄우기

<script type="text/javascript">
   	var popup = N().popup({
   		url: "html/indx/header.html",
   		title: "Global Navigation Bar",
   		width: 800,
   		draggable: true
   	});
   	popup.open();
</script>

4. 다른 (블록)페이지를 팝업으로 띄우기(상태가 유지 되지 않고 매번 팝업을 다시 불러옴)

<script type="text/javascript">
   	N().popup({
   		url: "html/indx/header.html",
   		title: "Global Navigation Bar",
   		width: 800,
   		closeMode: "remove" // "hide" 는 팝업의 상태를 유지 하고 "remove" 는 팝업을 닫을 때 마다 팝업 요소를 제거 함.
   	}).open();
</script>

5. 팝업에 데이터 전달하고 팝업으로부터 데이터 받기

5.1. 팝업페이지(popup.html)

<!-- Style -->
<style>
	.POPUP01 #result { width: 400px; height: 300px; border: 1px solid #4CAF50; }
</style>

<!-- View -->
<article class="POPUP01" title="onOpen Example">
	<p id="result"></p>
	<button id="close">닫기</button>
</article>

<!-- Controller -->
<script type="text/javascript">
N(".POPUP01").cont({
    init : function(view, request) {
    	var caller = this.caller; // caller는 현제 팝업페이지를 열어준 N.popup 인스턴스
    	N("#close", this.view).click(function() { // 닫기버튼에 이벤트 바인드
    		caller.close("Hello~ parent's Opener"); // close 함수의 첫번째 인자로 onOpenData 데이터 전달
    	});
    },
    // 팝업이 열릴때마다 실행될 함수
    onOpenFunc : function(onOpenData) {
    	N("#result", this.view).text(onOpenData);
    }
});
</script>

5.2. 부모페이지(parent.html)

<!-- Style -->
<style>
	.POPUP01 #result { width: 400px; height: 300px; border: 1px solid #4CAF50; }
</style>

<!-- View -->
<article class="POPUP02" title="onOpen Example">
	<p id="result"></p>
</article>

<!-- Controller -->
<script type="text/javascript">
N(".POPUP02").cont({
    init : function(view, request) {
	   	var popup = N().popup({
	   		url: "./popup.html",
	   		onOpen: "onOpenFunc", // 팝업이 열릴때마다 실행될 함수명
	   		onClose: function(onCloseData) { // 팝업이 닫힐때마다 실행 될 onClose 이벤트 핸들러
	   			N("#result", this.view).text(onCloseData);
	   		}
	   	});
	   	popup.open("Hello~ popup's onOpen!"); // open 함수의 첫번째 인자로 onOpenData 데이터 전달
    }
});
</script>
