생성자 Constructor

Name Arguments Type Return Description
N.cont N/A N/A (Controller)callback

Controller를 초기화 합니다.

N.cont는 컨트롤러 클래스를 등록하고 관리 해 주는 역할을 수행 합니다.
obj jquery object N/A View 영역을 selector로 지정 합니다.
callback json object class N/A Controller 클래스를 등록 합니다.
N(selector).cont N/A N/A N/A

N.cont의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

Create an object instance of N.cont with jQuery plugin method.

var docs = N(context).docs(opts);

객체 인스턴스를 생성하는 방식은 다르지만 new N.cont() 로 생성한 인스턴스와 N().cont 로 생성한 인스턴스는 동일 합니다. N() 함수의 첫번째 인자가 new N.cont 생성자의 첫 번째 인자로 설정 됩니다.

The method of creating object instances is different, but the instance created with newcont and the instance created with cont are the same. The first argument of the N () function is set as the first argument of the new N.cont constructor.
callback object Class N/A N.cont 의 callback 인자(argument) 와 같습니다.
