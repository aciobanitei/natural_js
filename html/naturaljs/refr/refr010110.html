N.message 객체의 함수 Function of N.message Object

"N.message.아래함수명(arg[0...N])" 과 같이 사용하면 됩니다.

Name Arguments Type Return Description
get N/A N/A N/A 메시지 리소스(config 에 정의)에서 현재 로케일에 맞게 메시지를 가져옵니다.
resource object N/A

메시지 리소스 객체를 지정 합니다.

메시지 리소스는 object 타입으로 만들면 되고 다음과 같이 메시지 세트를 구성 해야 합니다.

var message = {
    "ko_KR" : {
        key : ＂메시지{0}＂
    },
    "en_US" : {
        key : ＂message{0}＂
    }
}

var msg = N.message.get(message, "key", [" 입니다"]);

결과 : msg = "메시지 입니다"

key string N/A 메시지 키
vars array N/A 메시지 변수를 치환할 값
