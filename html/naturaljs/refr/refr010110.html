N.message 객체의 함수 Function of N.message Object

"N.message.아래함수명(arg[0...N])" 과 같이 사용하면 됩니다.

Name Arguments Type Return Description
get N/A N/A N/A

입력 된 메시지 리소스에서 N.config에 설정 된 로케일에 맞는 메시지를 반환 해 줍니다.

프레임웍의 기본 로케일은 N.config(natural.config.js) 의 N.context.attr("core").locale 속성값으로 정의 할 수 있고 N.locale 함수로 로케일 값을 가져오거나 변경 가능 합니다.
resource object N/A

메시지 리소스 객체를 지정 합니다.

Specifies the message resource object.

메시지 리소스는 object 타입으로 생성하고 다음과 같이 로케일별 메시지 세트를 구성 해야 합니다.

Message resource must be created as object type and the message set for each locale should be composed as follows.

var message = {
    "ko_KR" : {
        key : "안녕 {0}."
    },
    "en_US" : {
        key : "Hello {0}."
    }
}

var msg = N.message.get(message, "key", ["Natural-JS"]);

// msg : "Hello Natural-JS."
key string N/A

메시지 프로퍼티명.

Message property name.
vars array N/A

메시지의 변수를 입력 한 값으로 치환 해 줍니다.

The variable of the message is replaced with the entered value.

메시지에 선언 된 {index} 와 같은 변수가 vars 옵션으로 설정 한 array 의 index 에 해당 하는 값으로 치환 됩니다.

Variables such as {index} declared in the message are replaced with the value corresponding to the index of the array set with the vars option.
