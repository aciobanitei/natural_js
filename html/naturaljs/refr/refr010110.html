N.message 객체의 함수 Function of N.message Object

"N.message.아래함수명(arg[0...N])" 과 같이 사용하면 됩니다.

Name Arguments Type Return Description
get N/A N/A N/A

입력 된 메시지 리소스에서 프레임웍에 설정 된 로케일에 맞는 메시지를 반환 해 줍니다.

프레임웍의 기본 로케일은 N.config(natural.config.js) 의 N.context.attr("core").locale 속성값으로 정의 할 수 있고 N.locale 함수로 로케일 값을 가져오거나 변경 가능 합니다.
resource object N/A

메시지 리소스 객체를 지정 합니다.

메시지 리소스는 object 타입으로 만들면 되고 다음과 같이 메시지 세트를 구성 해야 합니다.

var message = {
    "ko_KR" : {
        key : ＂메시지{0}＂
    },
    "en_US" : {
        key : ＂message{0}＂
    }
}

var msg = N.message.get(message, "key", [" 입니다"]);

결과 : msg = "메시지 입니다"

key string N/A 메시지 키
vars array N/A 메시지 변수를 치환할 값
