생성자 Constructor

Name Arguments Type Return Description
N.formatter N/A N/A N.formatter

N.formatter 인스턴스를 생성합니다.

Create an N.formatter instance.

var formatter = N.formatter(data, rules);
data object N/A 서식화 할 데이터를 지정 합니다. 데이터는 array[object] 타입으로 입력 해야 합니다.
rules object|jquery object(elements) N/A

룰은 다음과 같이 두가지 타입으로 지정 가능 합니다.

* 라이브러리 초기화 시 명령형 옵션으로 지정 : 

// { "컬럼명" : [[룰명, arg0, arg1...], [룰명, arg0, arg1...]] }
{
    "numeric" : [["trimtoempty"], ["numeric", "#,###.##0000"]],
    "generic" : [["trimtoempty"], ["generic", "@@ABCD"]],
    "limit" : [["trimtoempty"], ["limit", "13", "..."]],
    "etc" : [["date", 12]]
}

* data-format 속성에 선언형 옵션으로 지정 : 

<input id="limit" type="text"
    data-format='[["trimtoempty"], ["limit", "13", "..." ]]' />
<!-- [ ["룰명", "룰arguments[0]", "룰 arguments[1]"], ... ] -->

선언형으로 포멧 룰 지정 시 동적으로 포멧 룰을 변경 하려면 다음과 같이 해당 입력 요소에 "format" 이라는 속성에 담겨있는 data(rules)를 변경 하면 됩니다.

N("#limit").data("format", [["trimtoempty"]])
N(obj).formatter N/A N/A N.formatter

N.formatter의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

Create an object instance of N.formatter with jQuery plugin method.

var formatter = N(data).formatter(rules);

객체 인스턴스를 생성하는 방식은 다르지만 new N.formatter() 로 생성한 인스턴스와 N().formatter 로 생성한 인스턴스는 동일 합니다. N() 함수의 첫번째 인자가 new N.formatter 생성자의 첫 번째 인자로 설정 됩니다.

The method of creating object instances is different, but the instance created with newformatter and the instance created with formatter are the same. The first argument of the N () function is set as the first argument of the new N.formatter constructor.
rules object|jquery object(elements) N/A

N.formatter 함수의 두번째 인자(rules)와 같습니다.

Same as the second argument(rules) of the N.formatter function.
