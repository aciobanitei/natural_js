N.browser 객체의 함수 Function of N.browser Object

"N.browser.아래함수명(arg[0...N])" 과 같이 사용하면 됩니다.

Name Arguments Type Return Description
cookie N/A N/A undefined 쿠키를 생성 합니다.
name string N/A 쿠키명
value string N/A 쿠키값
expiredays number N/A 만료일
domain string N/A 쿠키가 생성될 도메인
removeCookie N/A N/A undefined 생성된 쿠키를 제거 합니다.
name string N/A 쿠키명
domain string N/A 해당 쿠키가 생성된 도메인
msieVersion N/A N/A number 현재 접속한 Microsoft Internet Explorer(이하 MSIE)의 버전을 가져옵니다. MSIE가 아닐 경우 0을 반환 합니다.
is N/A N/A boolean 입력한 브라우저명과 현재 접속한 브라우저가 일치하면 true를 반환 합니다.
name string N/A 브라우저 명(opera, firefox, safari, chrome, ie) 나 모바일 OS 명(android, ios)
contextPath N/A N/A number 현재 접속한 브라우저 URL 에서 컨텍스트 경로를 반환 합니다.
scrollbarWidth N/A N/A number 브라우저의 스크롤바의 넓이를 반환 합니다.
