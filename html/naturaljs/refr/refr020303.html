생성자 Constructor

Name Arguments Type Return Description
N.comm N/A N/A N.comm

N.comm 인스턴스를 생성합니다.

Create an N.comm instance.

var comm = N.comm(param, opts|url);
obj jquery object|string|array|object N/A

jquery object(selector|element) 타입의 인자로 지정하고 두 번째인자로 특정 페이지의 URL을 지정하면 가져온 HTML 컨텐츠를 지정한 요소 안에 넣어줍니다.

json object 타입의 인자로 지정하면 Communicator.request 의 옵션으로 지정 됩니다.

jquery object(json object) 타입의 인자로 지정하면 지정한 값을 서버로 전송한 파라미터로 인식하여 서버에 요청을 보낼 때 전송 해 줍니다.

  • html page 요청 : 

    N.comm(N("#contents"), "page.html").submit();

// 페이지 파라미터 전송(불러오는 페이지의 Controller object 의 init 함수의 request 인자(arguments[1])로 받을 수 있음)
N.comm(N("#contents"), "page.html").request.attr("pageParam", "value").submit();

  • JSON data 요청 : 

    // obj 인자가 string 이면 url 옵션으로 설정 됩니다.
N.comm("data.json").submit(function(data) {});

// obj 인자가 array 나 object 이면 요청 파라미터로 설정 됩니다.
N.comm({ "param" : "value" }, "data.json").submit(function(data) {});

// url 인자가 object 이면 기본 옵션으로 설정 됩니다.
N.comm({ "param" : "value" }, {
    url : "data.json"
}).submit(function(data) {});

인자가 파라미터로 작동할 경우에는 Communicator.request.options 의 data 속성으로 저정되어 전달 됩니다.

파라미터는 요청 타입(type, HTTP 에서는 Method)이 GET 일 경우 지정한 파라미터 타입이 JSON 문자열 이나 array[json object] 문자열 이면 q 라는 파라미터 변수값으로 해당 문자열을(URL Encode 함) 지정 합니다.
예) text.html?q=[{"a" : 1}, {"a" : 2}]
브라우저마다 GET 파라미터로 보낼수 있는 양이 정해져 있어 파라미터가 잘릴 수 있으니 브라우저별 GET 파라미터 최대 전송량을 파악하여 사용하기 바랍니다.

opts|url object|string N/A

string을 입력하면 Communicator.request 객체의 기본 옵션 중 url옵션으로 설정 됩니다.

object를 입력하면 Communicator.request 객체의 기본 옵션으로 설정 됩니다.
N(obj).comm N/A N/A N.comm

N.comm의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

Create an object instance of N.comm with jQuery plugin method.

var comm = N(param|element).comm(opts|url);

객체 인스턴스를 생성하는 방식은 다르지만 "new N.comm()" 로 생성한 인스턴스와 "N().comm" 로 생성한 인스턴스는 동일 합니다. N() 함수의 첫번째 인자가 new N.comm 생성자의 첫 번째 인자로 설정 됩니다.

The method of creating object instances is different, but the instance created with "new N.comm()" and the instance created with "N().comm" are the same. The first argument of the N () function is set as the first argument of the new N.comm constructor.

  • html page 요청 : 

    N("#contents").comm("page.html").submit();

// 페이지 파라미터 전송(불러오는 페이지의 Controller object 의 init 함수의 request 인자(arguments[1])로 받을 수 있음)
N("#contents").comm("page.html").request.attr("pageParam", "value").submit();

  • JSON data 요청 : 

    N({ "param" : "value" }).comm("data.json").submit(function(data) {});
// comm 메서드의 첫번째 인자가 object 이면 기본 옵션으로 설정 됩니다.
N({ "param" : "value" }).comm({
    url : "data.json"
}).submit(function(data) {});
opts|url object|string N/A

N.comm 함수의 두번째 인자(opts|url)와 같습니다.

Same as the second argument(opts|url) of the N.comm function.
