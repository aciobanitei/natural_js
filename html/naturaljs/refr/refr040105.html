Methods

Name Arguments Type Return Description
context - - jquery object 컴포넌트를 적용한 요소를 반환 합니다.
selector string - selector 문자열을 입력하면 context 옵션으로 지정한 요소에서 선택된 요소를 반환 합니다.
show - - N.alert 메시지 대화상자를 보여 줍니다.
hide - - N.alert 메시지 대화상자를 숨깁니다.
remove - - N.alert 메시지 대화상자를 표현하는 HTML Element들을 제거합니다.

"ESC" 키를 눌러도 팝업이 remove 됩니다. N.alert 라이브러리를 공유하는 다른 컴포넌트(N.popup, N.grid(Validate), N.form(Validate))들도 이와 같은 동작을 합니다.
