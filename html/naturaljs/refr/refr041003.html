생성자 Constructor

Name Arguments Type Return Description
N.pagination N/A N/A N.pagination N.pagination의 객체 인스턴스를 생성합니다.
data array[json object] N/A 기본옵션의 data 옵션과 같습니다.
opts object | jQuery Selector(string | jquery object) N/A

기본 옵션을 object 타입으로 지정 합니다.

jQuery Selector 를 입력하면 입력한 인자가 context 옵션으로 지정 됩니다.
N(obj).pagination N/A N/A N.pagination

N.pagination의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

N.pagination과 동작되는 로직은 같으며 내부적으로 N()의 첫번째 인자가 N.pagination()의 첫 번째 인자로 지정 됩니다.
opts object | jQuery Selector(string | jquery object) N/A N.pagination의 opts 옵션과 같습니다.
